## Dashboard branding assets

Place your logo files in this folder so `dashboard_index.html` can load them.

### Expected filenames

- `trifecta-logo.png` (Trifecta logo)
- `illuminate-logo.png` (Illuminate logo)

If the files aren’t present, the dashboard will show simple placeholder tiles instead.

### Recommended sizes

- PNG or SVG
- Square is best (e.g., 256×256 or 512×512)
- Transparent background preferred
